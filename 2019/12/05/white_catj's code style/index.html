white_catj's code style

首先，我决定改改码风始于这个博客开始创建

所以你能在以前的题解看到很多其它的码风，习惯也不大一样，本篇文章主要是记载现在的（想要改的）码风

大概是很像《算法进阶》的代码

举个例子吧

总结一下：

1.程序每块之间隔一行，如include等一块，每个函数一块，main一块

2.include下面是define，然后是using namespace std

3.定义变量和数组分行
？？
4.逗号，分号后面必有一个空格

5.等号（不等号）两边各一个空格

6.上大括号不换行

7.if或for等的语句下面如果只有一句，那么不换行

差不多就这么多，并且不会用什么endif之类的东西，也不打算用
